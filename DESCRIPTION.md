# Salesforce Experience Cloud PRM Sandbox

Salesforce Experience Cloud PRM Sandbox helps product managers, admins, and testers preview partner portal experiences without repeatedly logging in and out with different test users.

Open a Salesforce Experience Cloud sandbox or staging portal, then use the extension popup to create partner personas such as Tier 1 Reseller, Referral Affiliate, MSP, or regional partner variants. Each persona can store partner type, tier, and region values detected from the connected Salesforce environment.

Use Single View to apply one persona to the current tab. Use Split-Screen, Compare Three, or Compare Four to preview multiple personas side by side in the same browser tab. This makes it easier to compare visibility, page layout behavior, field differences, moderation rules, deal registration access, MDF visibility, and other partner-specific experiences.

To start, open your Salesforce Experience Cloud page, click the PRM Sandbox extension icon, choose or create a persona, and select the comparison mode you want to test.
