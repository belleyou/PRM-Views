# Privacy Policy for Live PRM Partner Experience Sandbox Modifier

Effective date: July 18, 2026

Live PRM Partner Experience Sandbox Modifier is a Chrome extension designed to help Salesforce Experience Cloud teams test sandbox and staging partner portal experiences.

## Data Stored Locally

The extension stores partner persona configuration in Chrome extension storage on the user's device. This may include persona names, partner type, partner tier, region, selected view mode, and the active persona.

## Salesforce Context

When the user chooses to run the extension on a Salesforce Experience Cloud sandbox or staging site, the extension may read the active tab URL to confirm the page is a supported Salesforce environment. It may also use the user's existing Salesforce browser session to support persona simulation or org-aware metadata detection when the Salesforce user has the required permissions.

## Data We Do Not Collect

The extension does not sell user data. The extension does not send user data to third-party analytics services. The extension does not store Salesforce passwords or credentials. The extension does not use a remote backend in this prototype.

## Permissions

The extension requests Chrome permissions only to support its core Salesforce testing workflow, including local storage, active tab access, content script injection, Salesforce session-aware testing, and optional access for enterprise custom Salesforce domains approved by the user.

## Contact

For questions about this privacy policy, contact the extension publisher through the Chrome Web Store listing or the GitHub repository for this project.
